# BlueArchive_Bot
Fetch data from Blue Archive websites via python, then control these data stored in the database by Discord bot.

Resource is from : https://bluearchive.wikiru.jp/?%E3%82%AD%E3%83%A3%E3%83%A9%E3%82%AF%E3%82%BF%E3%83%BC%E4%B8%80%E8%A6%A7

Current features :

    (General)
    ¡DConnect to the database.
    ¡DCheck, create the tables in the database file.
    ¡DDump the file fetched from the website using BeautifulSoup

    (Discord bot)
    ¡DLaunch as a Discord bot.
        (The login token is saved as an ignored json file for security.)
    ¡DSupport some slash commands such as greeting, speaking etc.
